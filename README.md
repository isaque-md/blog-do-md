# 🎖️ Blog Pessoal - Isaque Dantas

Projeto desenvolvido para a disciplina de Programação Web I, focado na criação de um blog pessoal utilizando tecnologias web fundamentais e auxílio de Inteligência Artificial.

## 🚀 Tecnologias Utilizadas
- **HTML5**: Estruturação de conteúdo.
- **CSS3**: Estilização com design responsivo.
- **JavaScript**: Efeitos dinâmicos (como o fundo estilo Matrix).
- **Git & GitHub**: Controlo de versão e alojamento.

## 🤖 Parceria com IA
Conforme solicitado na atividade, este projeto foi desenvolvido com o apoio da seguinte IA:
- **Google Gemini**: Auxiliou na estruturação do código, resolução de bugs de contraste no CSS, correção de problemas de envio (deploy) no Git e organização da documentação.
- A IA serviu de apoio técnico e para a aprendizagem, o Gemini não acertava 100% das vezes, o que me motivava a ir atrás dos erros e dos desenvolvimentos por conta própria.
